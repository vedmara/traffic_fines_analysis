# Анализ штрафов автопарка

Отчет по штрафам за период 6 месяцев.

# Цели анализа
- Оценка финансовых потерь от штрафов
- Выявление проблемных водителей и автомобилей
- Анализ времени и мест нарушений
- Подготовка отчета для руководства

# Источник данных
Выгрузка штрафов (даты, водители, автомобили, суммы, места нарушений)

# Результаты
- Сводные показатели
- Рейтинги водителей и ТС
- Временные и географические паттерны
